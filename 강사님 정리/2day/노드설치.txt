---------------------------------------------------------------------------
우분트 리눅스 업데이트
sudo apt-get update
---------------------------------------------------------------------------
노드 설치
// 설치 인식을 위해서 넣어줌
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
---------------------------------------------------------------------------
소스 업로드 (ftp) 
mkdir server를 한다.
/home/ubuntu/server/소스
---------------------------------------------------------------------------
모듈설치
cd server
npm install 
---------------------------------------------------------------------------
forever 설치 : 노드 백그라운드 구동 관리 설치
sudo npm install -g forever
---------------------------------------------------------------------------
구동 (뒤에 위치는 현제 프럼프트 위치에 따라 다름)
forever start ./server/bin/www
---------------------------------------------------------------------------
로그보기
forever list           => 로그파일 경로 확인
tail -f 로그파일경로   => 로그 확인
---------------------------------------------------------------------------
서버 중단
forever stopall        => 전체 프로세스 중단
forever stop id        => 특정 프로세스만 중단 (id는 forever list를 통해서 확인)
---------------------------------------------------------------------------
서버 재가동
forever restart ./server/bin/www
---------------------------------------------------------------------------
매주 월요일 새벽 5시 서버 재가동
0 5 * * 1
---------------------------------------------------------------------------
편집 (root 권한에서 수행 : sudo su)
crontab -e => 편집
crontab -l => 스케줄러 보기
crontab -r => 스케줄러 삭제
---------------------------------------------------------------------------
스케줄러 입력
crontab -e
---------------------------------------------------------------------------
# 1분마다 reboot.sh 가동, 로그는 crontab.log 여기에 남김
* * * * * /home/ubuntu/server/reboot.sh >> /home/ubuntu/server/log/crontab.log 2>&1
---------------------------------------------------------------------------

reboot.sh -----------------------------------------------------------------
sudo /usr/bin/forever restart ./bin/www

---------------------------------------------------------------------------

적절하게 풀어준다
